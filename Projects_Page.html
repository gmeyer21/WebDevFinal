Apache/Nginx Web Server on Raspberry Pi

I created a web server once using Apache and once using Nginx. When making the web server in Apache/Nginx I was running both systems on the Raspberry Pi, and configuring it all through SSH. These configurations included firewalls, cloudflared tunneling, SSL/TLS certificates, BasicAuth, and more. The Pi was separated on a VLAN and had an isolated port. This project had really taken me out of my comfort zone in terms of ability. I had to adapt and learn a great deal of Bash scripting and understanding of Linux based OS's. The Apache server was made on Ubuntu OS, and the Nginx server was made on FreeBSD OS.
